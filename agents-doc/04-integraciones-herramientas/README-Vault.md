# Guía Rápida - Vault de Metodología Agents-Doc

Bienvenido a tu vault de la metodología de trabajo con agentes de IA.

## Estructura del Vault

- **00-core/** → Archivos principales de la metodología (plantillas y principios)
- **01-plantillas/** → Plantillas listas para usar
- **02-guias/** → Guías detalladas (TDD, Recuperación, Eficiencia Avanzada, etc.)
- **03-roles-y-patrones/** → Matriz de roles y patrón Orchestrator
- **04-ejemplos/** → Ejemplos por stack
- **05-proyectos/** → Tus proyectos reales usando la metodología
- **06-canvas/** → Diagramas visuales de flujos (JSON Canvas profesional)
- **07-templates-templater/** → Templates listos para Templater (usa Ctrl+P → "Templates: Insert")
- **08-dashboards/** → Dashboard + **Obsidian Bases** (.base) dinámicos (Plantillas.base, Guias.base)
- **99-personal/Memory-Bank/** → **Memoria estructurada para agentes** (Executive Summary, Decisiones, Protocolo de Memoria, Session Logs)
- **99-personal/** → Tus notas y ajustes personales + Memory Bank del vault

**Objetivo actual del agente**: Generar documentación profesional completa de software (ver [[objetivo-agente-documentacion-software-profesional]] y la plantilla principal [[plantilla-generacion-especificacion-software-profesional]])

> [!note] Agent Skills instalados
> El vault tiene `kepano/obsidian-skills` en `.kilo/skills/`. El agente (Kilo) ahora entiende y puede crear/editar Markdown Obsidian, Bases, Canvas JSON y usar el CLI de Obsidian.

## Primeros Pasos Recomendados

1. Abre `08-dashboards/Dashboard.md`
2. Abre **[[08-dashboards/Plantillas.base]]** y **[[08-dashboards/Guias.base]]** (vistas modernas)
3. **Para agentes**: Lee inmediatamente `99-personal/Memory-Bank/Protocolo-Memoria-Agente.md` y `Executive-Summary.md`
4. Instala los plugins si aún no lo hiciste: Templater, Dataview, Obsidian Git
5. Configura Obsidian Git para backup automático
6. Prueba los templates de Templater (están en `07-templates-templater/`)
7. Abre `06-canvas/Orchestrator-Workflow.canvas` para el flujo visual actualizado
