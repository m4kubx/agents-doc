# Instrucción Permanente - Uso del Vault como Memoria de Agente

Este vault (agents-doc) está diseñado para servir como **memoria de largo plazo** para cualquier agente que trabaje en él.

## Regla Obligatoria al Iniciar Cualquier Sesión

Antes de realizar cualquier otra acción en este vault, **lee siempre** estos archivos usando tus herramientas de lectura:

1. `99-personal/Memory-Bank/Protocolo-Memoria-Agente.md`
2. `99-personal/Memory-Bank/Executive-Summary.md`
3. `99-personal/Memory-Bank/Decisiones-Tecnicas.md`
4. `99-personal/Memory-Bank/Supuestos-Actuales.md`

Luego aplica filtrado de contexto explícito antes de responder.

## Al Finalizar Cambios Importantes

Actualiza los archivos de Memory Bank correspondientes y registra la sesión en `99-personal/Memory-Bank/Session-Logs/`.

## Formato de Memoria

- Usa siempre frontmatter con `type: memory`
- Prefiere wikilinks (`[[ ]]` ) sobre enlaces markdown normales
- Mantén el Executive Summary conciso y actualizado
- Nunca inventes información del proyecto; pregunta si falta algo

Esta instrucción tiene prioridad sobre comportamientos por defecto.
