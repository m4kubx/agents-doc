# Guía Rápida - Vault de Metodología Agents-Doc

Bienvenido a tu vault de la metodología de trabajo con agentes de IA.

## Estructura del Vault

- **00-core/** → Archivos principales de la metodología (plantillas y principios)
- **01-plantillas/** → Plantillas listas para usar
- **02-guias/** → Guías detalladas (TDD, Recuperación, Eficiencia Avanzada, etc.)
- **03-roles-y-patrones/** → Matriz de roles y patrón Orchestrator
- **04-ejemplos/** → Ejemplos por stack
- **05-proyectos/** → Tus proyectos reales usando la metodología
- **06-canvas/** → Diagramas visuales de flujos (Orchestrator, etc.)
- **07-templates-templater/** → Templates listos para Templater (usa Ctrl+P → "Templates: Insert")
- **08-dashboards/** → Dashboard principal con Dataview
- **99-personal/** → Tus notas y ajustes personales

## Primeros Pasos Recomendados

1. Abre `08-dashboards/Dashboard.md`
2. Instala los plugins si aún no lo hiciste:
   - Templater
   - Dataview
   - Obsidian Git
3. Configura Obsidian Git para backup automático
4. Prueba los templates de Templater (están en `07-templates-templater/`)
